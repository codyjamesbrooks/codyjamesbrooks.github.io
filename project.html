Machine Learning with Swedish Insurance Claims

The idea behind this project was to be able to predict the total value that a certain Swedish auto insurance company could expect to pay knowing how many claims were filled over a given month. I used multiple machine learning techniques during this project such as KNearestNeighbors and Logistic Regression. I also created a number of graphs using Matplotlib that help visiualize the data as well as the Machine Learing process. I am still adding to this project so stay tuned for updates.
